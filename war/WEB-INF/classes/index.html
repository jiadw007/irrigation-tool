This web tool provides you with the ability to test different types of irrigation technologies in a virtual environment.

Clicking on each of the examples below to see a picture and description of each type of irrigation.

Time-Based With a Soil Moisture Sensor

More Information

Time-Based With a Soil Moisture Sensor Information

Uses an automatic timer with a soil moisture sensor. The sensor measures soil moisture and reports to the controller if too wet; if too wet, irrigation does not occur.

Time-based with soil moisture sensor irrigation resources

Evapotranspiration Controllers

More Information

Evapotranspiration Controllers Information

Uses weather measurements to estimate evapotranspiration. The estimated evapotranspiration losses are used to determine an irrigation schedule.
